INFINITY WAR

Avengers's Trivia Game

There are 10 questions for player to select who is the actor of character in the Avengers Infintity War.
Click Menu - Start to start the game!!!

GAME RULE

This trivia game will show only one question until the player answers it or their time runs out (after 15 sec).

If the player selects the correct answer, it shows a screen congratulating message for choosing the right option. After a few seconds, display the next question automatically.

If the player runs out of time, it will tell the player that time's up and display the correct answer. Wait a few seconds, then show the next question.

If the player chooses the wrong answer, it will tell the player that they selected the wrong option and then display the correct answer. Wait a few seconds, then show the next question.

On the final screen, it shows the number of correct answers, incorrect answers, and an option to restart the game (without reloading the page).
